My Best Project

When I say my best project. It's a project that I not only learned a lot while I was developing it, but enjoyed writing every line of code. Now, it wasn't all sunshine & rainbows. I might've lost my mind one or two times while making it, but it's a production core used on Minecraft servers. This core tracks server data, player ranks, player metadata, accounts related to other accounts, and much more. It also has an adaptive logging system with many implementations if you want to integrate it with a website. This project integrates the Minecraft API, Mongo Database, Redis Pub Subs, JDA, and Minecraft Velocity.
